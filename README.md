# 💰 Premium Budget Planner

A professional personal finance dashboard to track income, expenses, and savings goals.

## ✨ Features

- 📊 **Real-time Analytics** - Automatic calculation of totals and trends
- 💰 **Income/Expense Tracking** - Categorize all transactions
- 🎯 **Savings Goals** - Set targets and track progress
- 📈 **Category Breakdown** - Visual charts for spending analysis
- 🔍 **Advanced Filtering** - Search by date, category, or type
- 💾 **Export/Import** - Backup data as CSV or JSON
- 🌙 **Premium Dark Theme** - Glassmorphism design

## 🛠️ Technologies Used

- React.js 18
- Custom CSS (Glassmorphism)
- localStorage API
- CSV/JSON Export

## 🎯 How to Use

1. Click "Add Transaction" to add income or expenses
2. Set your savings goal
3. Use filters to analyze spending
4. Switch to Charts view for visual insights
5. Export data for record keeping
